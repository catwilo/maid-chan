# maid — context
## status: STABLE

Trash and zsh history manager. POSIX sh. No external deps except fzf (history search only).

## structure
- maid.sh          — entrypoint, dispatches subcommands
- lib/trash.sh     — trash/restore/list/empty; metadata via .tenshi-sama companion files
- lib/history.sh   — dedup (backs up to $HISTFILE.bak before edit) + fzf search
- install.sh       — symlinks maid.sh to ~/.local/bin/maid (idempotent)
- .gitignore       — excludes *.log *.bak *.swp *~ .DS_Store

## config
- MAID_TRASH  — trash dir (default: ~/.Maid-Trash, outside repo by design)
- HISTFILE    — zsh history (default: ~/.zsh_history)

## alias
- zsh-setup/dotfiles/.addons-zsh/aliass/shared.zsh: alias maid='~/unix-toolkit-tools/maid-chan/maid.sh'

## last-known-good
